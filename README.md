## Description 
Application web permettant de suivre une routine matinale guidée, composée d’exercices chronométrés, d’instructions interactives et d’une interface fluide animée via GSAP.

Actuellement, ce projet est déployé grâce à Vercel.

### Usage  
* Lancez votre routine du matin.
* Suivez les exercices un par un, accompagnés d’un minuteur circulaire animé.
* Affichez les instructions détaillées à tout moment via le tiroir d’information.
* Progressez automatiquement jusqu’à la fin de la séance.
